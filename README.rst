======
Marvin
======

Our robot for template creation, works with `Cookiecutter <https://cookiecutter.readthedocs.io/en/latest/index.html>`_.

Features
========

- Creates templates, according to our setup and style-guide

Dependencies
============

- `Cookiecutter <https://cookiecutter.readthedocs.io/en/latest/index.html>`_.

Installation
============

None

Usage
=====

None official

.. code-block:: shell

   /home/svx/bin/tools/bin/cookiecutter cookiecutter-rakpart/
